## Run Locally

Clone the project

```bash
  git clone https://github.com/samnoon1971/calculator-k12
```

Go to the project directory

```bash
  cd calculator-k12
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
